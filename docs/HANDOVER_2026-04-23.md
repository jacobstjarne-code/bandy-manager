### Skottbild — komplett redesign

**Commit `9a75986`** — design: skottbild — två isolerade straffzoner istf ett plan

Skottbilden föreställde ett fullständigt plan men var inte i skala, vilket fick mittlinjen och centrala geometrin att se förvirrande ut. Lösningen: rendera **två isolerade straffzoner** med en separator emellan — ingen skalpretension.

**Geometriförändringar:**
- Varje zon: mål (näthinten + ribbstolpe via linjer), målområde (litet rektangel), straffområde (stort rektangel), straffpunkt, straff-D-båge
- Top-D: `M 121 50 A 25 25 0 0 1 159 50` — sweep=1 (medurs) böjer BORT från topp-målet (nedåt) ✓
- Botten-D: `M 121 160 A 25 25 0 0 0 159 160` — sweep=0 (moturs) böjer BORT från botten-målet (uppåt) ✓
- Straffpunkter beräknade korrekt (30px från respektive mållinje, bågcenter = straffpunkt, r=25)

**Bort:**
- Mittlinje (förvirrande, plan ej i skala)
- Center-ellips (hörde inte hemma)
- Fyllda rektanglar för mål → ersatta av linjer (crossbar + stolpar)

**Prickpositioner** (nextPos/nextOppPos) uppdaterade till nya zon-koordinater — prickar begränsas nu till sin respektive zon (y 0-100 topp, y 110-210 botten).

**Städat:** Dubbeldeklarationen `oppGoals`/`oppGoalCount` (identisk beräkning på rad 645 och 662) konsoliderad till `oppGoals`.
