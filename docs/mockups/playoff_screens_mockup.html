Designregler

• Helskärm — ingen GameHeader, ingen BottomNav.

• Max två card-sharp sektioner + en card-round stämningstext.

• Managed club = rgba(196,122,58,0.06) bakgrund, accent text.

• Vinnare = font-weight 700, text-primary. Förlorare = text-muted. INTE var(--success).

• Serieresultat = matcher vunna (3–1), INTE individuella matchresultat.

• Hero = Georgia 20-22px 800. Undertext = 12px text-secondary.

• CTA längst ner: margin-top auto.

• QF-resultat: enkla rader med border-bottom. INTE separata kort per serie.

• Ingen redundant "{winner} gick vidare"-text under serier.
